# 在行分析报告

## 报告目的
尝试从数据之中分析或挖掘中在行的特征和内在联系。包括产品本身的特点，以及特点对在行的影响；在行所属市场的特征；在行上供给方和需求方的特点；在行本身的运营特征等等。

尝试依据上述发现，对在行做出客观判断；再根据在行的优劣势，对在行过往的表现做出评价，并对未来的趋势做出预测。

最后尝试对在行的发展和决策提出现实可行的建议。
